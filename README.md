# Python Learning Platform

React + Vite learning platform project with lessons, quizzes, and progress tracking UI components.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to GitHub Pages

This repo is configured to deploy automatically from the `main` branch using GitHub Actions.

1. Open your repository on GitHub.
2. Go to `Settings` -> `Pages`.
3. Under `Build and deployment`, set `Source` to `GitHub Actions`.
4. Push to `main` (or run the workflow manually from the `Actions` tab).

Site URL:

`https://supranjal.github.io/python_learn/`
